# v0.6.0 — 세션 내용 요약 + 클린 설치/aimm init

릴리스: 2026-06-22 (KST)

## 한 줄 요약
파서가 그동안 버리던 **세션 대화 내용**(tool_use·사용자 프롬프트)을 결정적 닫힌-어휘 다이제스트로 추출해, analyze/portrait/주간 내러티브에 **"이 기간에 무엇을 했나"**(활동 믹스·다룬 영역·명령 카테고리·대화 깊이)를 넣는다. 더불어 GitHub 클론만으로 MCP 설치가 끝까지 동작하도록 `prepare` 빌드훅 + `aimm init` 원커맨드 셋업을 추가했다.

## 무엇이 바뀌었나
- **세션 내용 요약(신규)**: 메트릭은 `message.usage`만 읽고 본문을 버렸는데, 이제 같은 단일 패스에서(가드 위로 hoist) `message.content`를 스캔한다. 신규 `src/core/content.ts`가 도구→활동(구현/탐색/실행·검증/계획·조율/웹), 확장자→영역(TypeScript/Java/문서/설정…), 명령 첫 토큰→카테고리(버전관리/패키지/실행·테스트/파일/네트워크)로 결정적 롤업.
- **프라이버시 = 구성상 보장**: 다이제스트는 **닫힌 어휘 카운트만** 보관 — 도구 레지스트리명 / 알려진 확장자∪`기타` / 허용목록 동사∪`기타`. **원시 경로·명령 문자열·프롬프트 텍스트는 분류 후 폐기되어 파서를 떠나지 않는다.** narrative 사실줄도 카테고리·정수뿐이라 마스킹은 심층방어.
- **내용-미파악 격리**: analyze가 **cost-known(Claude Code) 세션의 `.content`만** 롤업 → Cursor 등 cost-unknown 소스는 `.content`가 있어도 제외(cost-unknown 격리의 평행). standup·hook·metrics 출력 불변.
- **클린 설치(핵심 버그픽스)**: `dist/`가 gitignore인데 `bin`이 `./dist/cli.js`라 클론 후 빌드 없이는 MCP 등록이 실패했다. `package.json`에 `"prepare": "npm run build"` 추가 → `npm install`만으로 자동 빌드.
- **`aimm init [--dry-run]`(신규)**: `~/.claude/settings.json`에 SessionEnd hook을 **안정 센티넬 기반 멱등** 병합(경로가 바뀌어도 중복 없이 교체)하고, `claude mcp add --scope user`를 시도(부재/실패/타임아웃 시 `.mcp.json` 폴백). 백업·딥머지(우리 키만)·`stdin:ignore`+타임아웃·복구 절차 출력.

## 출력 before/after
**before** — analyze는 모델·비용·시간대·프로젝트만. "무엇을 했나"는 `--repo` 커밋 타입에서만 와서 커밋 없는 세션(리서치·디버깅·설계)은 안 보였다.

**after** — 세션 내용에서 직접(이 릴리스 작업일 실데이터, 2026-06-22):
```

- 대화 깊이: 사용자 요청 ~19건 · 내용 분석된 세션 10건
```

- portrait에는 카운트·경로·프로젝트명 없이 추상 라벨만(`## 무엇에 썼나`). 주간 내러티브 사실줄엔 `[작업내용] 활동 …% · 영역 … · 명령 … · 요청 N건`.

## 이 버전을 만든 과정 (AI 페어링 기록)
브레인스토밍 → **적대적 spec 리뷰(3 리뷰어 워크플로, blocker 5건)** → TDD 11태스크 구현. 리뷰가 잡은 핵심 결함을 코드 전에 교정:
1. 파서 루프 재구성 — `role!==assistant`·`usage===undefined` 가드가 user 프롬프트·usage 없는 tool_use를 가로채 `userPrompts`가 항상 0이 될 뻔(가드 위로 hoist).
2. `commandVerbs` 원시 토큰 누출 차단(경로/스크립트명 가능 → 닫힌 어휘로).
3. 렌더 예시는 허용목록 동사만·`기타`는 카운트만, 내용 섹션/portrait에 `/`·`\` 부재 단언.
4. `exactOptionalPropertyTypes` — 신호 없으면 `content` 속성 생략.
5. `aimm init` 멱등을 볼라타일 절대경로가 아닌 **안정 센티넬**로.

## 알려진 한계
- **주제(subject) 미파악**: 무엇을 *다뤘나*(파일·도구 종류)는 잡지만 *무슨 주제*였는지는 아니다(자유텍스트 추출은 보류).
- **Claude Code 단독**: Cursor 등은 내용 미파악(어댑터가 내용을 신뢰 제공할 때 확장).
- **서브에이전트 비대칭**: 메인 세션의 디스패치는 세지만 서브에이전트 내부 도구 활동은 제외 → 무거운 서브에이전트 사용은 과소 보고.
- `aimm init` 폴백 `.mcp.json`은 프로젝트 스코프 — 전역은 출력된 `claude mcp add --scope user` 명령으로.
